Hilfe - Ansicht¶

Die Hilfeansicht ist diese Sphinx-Dokumentation die um eine BEdienerdokumentation erweitert wurde. Sie lässt sich über File -> Help aufrufen. Die entsprechenden HTML-Dokumente werden über eine QWebEngine gerendert. Das darüberliegende Fenster etc. ist in der Datei ‘Help.qml’implementiert:

 1import QtQuick 2.9
 2import QtQuick.Controls 2.5
 3import QtQuick.Controls.Material 2.15
 4import QtQuick.Layouts 1.3
 5import QtQuick.Window 2.15
 6import QtWebEngine 1.1
 7
 8
 9Window {
10    id: window
11    width: 640
12    height: 480
13    visible: true
14    title: "Help Menu"
15    WebEngineView {
16        id: webView
17        //url: "http://www.example.com"
18        anchors.fill: parent
19
20    }
21    Component.onCompleted: {
22            var path = Qt.resolvedUrl("../../sphinx_build/index.html")
23            webView.url = path
24    }
25}
