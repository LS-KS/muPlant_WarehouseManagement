RFID Server¶

RFID Server wird benötigt um die RFID Reader der Firma Feig Electronic GmbH anzusteuern. Dabei werden alle RFID Reader der µPlant über Modbus ausgelesen und die ausgelesenen Daten visualisiert. Die Daten werden außerdem über OPCUA im ganzen Netz zur Verfügung gestellt.

Das Rfid Server Plugin wird im Programm über die Menubar aufgerufen: Tools -> RFID Server. Für die Darstellung sind zwei QML Dateien zuständig. RFIDServerPlugin.qml rendert das Hauptfenster. Es enthölt eine ListView, die die Daten des QAbstractListModel über das QSortFilterProxyModel darstellt. Die Daten werden in dem Delegate RfidDelegate.qml dargestellt.
