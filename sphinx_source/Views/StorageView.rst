Lageransicht
============

Dieses QML Item definiert wie die Daten aus dem Datenmodell des Lagers gerendert werden. 
Die Daten werden in einer Tabellenansicht gerendert, dessen Zellen je eine `ProductView.qml` ist.
Die komplette Lageransicht ist in der Datei `StorageView.qml` definiert.
